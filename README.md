# Retail-Sales-Data-Agent
#The Objective
Build a Retail Sales Data Agent on Databricks using the provided retail sales
dataset. Your agent must help a business user, manager, or CEO ask questions
about a shop's sales performance and receive useful business insights — in plain
language, backed by real data.
End-to-end Retail Sales Data Agent that cleans, analyzes, and queries retail sales data to uncover sales trends, product performance, customer behavior, and revenue insights using Databricks, SQL, and AI.
